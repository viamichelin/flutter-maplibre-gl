iosLongClickDuration

iosLongClickDuration property

Duration? iosLongClickDuration
final

How long a user has to click the map on iOS until a long click is registered. Has no effect on web or Android. Can not be changed at runtime, only the initial value is used. If null, the default value of the native MapLibre library / of the OS is used.
